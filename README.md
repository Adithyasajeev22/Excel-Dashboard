# 📊 Excel Dashboard Projects

This repository contains interactive Microsoft Excel dashboards created using Excel features such as Pivot Tables, Pivot Charts, Data Cleaning, and Data Visualization techniques. These dashboards provide meaningful insights from different datasets.

---

## 🚀 Projects Included

### ✈️ Airline Dashboard

The Airline Dashboard analyzes airline-related data and provides insights into:

- Airline-wise price distribution
- Average ticket price
- Average duration
- Total price and total duration
- Duration vs Price analysis
- Arrival time distribution
- Departure time distribution
- Arrival time vs Duration relationship



---

### 🏠 Bengaluru House Data Dashboard

The Bengaluru House Data Dashboard provides insights into real estate trends and includes:

- Area type distribution
- Area type vs Balcony analysis
- Size vs Total Square Feet
- Bath vs Price relationship
- Price vs Location comparison
- Size vs Balcony analysis


---

### 🍽️ Zomato Dashboard

The Zomato Dashboard provides insights into restaurant and delivery data:

- Online delivery availability
- Table booking statistics
- Price range distribution
- City-wise price range analysis
- Delivery status analysis
- Aggregate Rating vs Votes
- Currency vs Price range
